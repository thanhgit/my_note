# How to use terminator terminal

## Open new tab
```bash
Ctl + Shift + T
```

## 

## Split horizontal
```bash
Ctl + Shift + O
```

## Split vertial
```bash
Ctl + Shift + E
```

## Edit terminal title
```bash
Ctl + Alt + X
```

## Edit window title
```bash
Ctl + Alt + W
```

## Move terminal
```bash
Alt + arrow
```

## Move tab
```bash
Ctl + PgUp/ PgDn
```
