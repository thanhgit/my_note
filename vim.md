# Vim editor

## Set number 
```bash
:set number
:set number!
```
## Set default number of columns used for the line number
```bash
:set numberwidth=10
```
## Spell checking your language
```bash
:set spell
:set spell!
```

## Faster navigation in a file
- Move begin and end in file
```bash
{ : begin

} : end
```
- Move begin and end in setense
```bash
( : begin

) : end
```
